# github-branch-policy

Branch governance and workflow-hygiene audit skill for GitHub repositories.

## Files

- `SKILL.md`: full audit workflow, verification commands, and remediation steps
- `agents/openai.yaml`: UI metadata for skill discovery

## Scope

This skill focuses on:
- Rulesets and required status checks
- Solo-dev review policy settings
- Delete-on-merge and cleanup workflows
- Auto-merge reliability and workflow registration hygiene
